# 基本概念
- 贪心算法的适用条件：要求问题具有贪心选择性质和最优子结构性质
    - 贪心选择性质：整体最优解可以通过一系列局部最优的选择（贪心）达到
    - 最优子结构性质：一个问题的最优解包含其子问题的最优解
- 贪心算法的操作要求：
    - 符合问题要求
    - 符合局部最优
    - 执行后不能撤回
- 证明：
    - 尝试证明贪心得到的解与最优解是相同的
